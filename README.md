# dev-dylan-cli-hello-world

## Description
A POC for dylan programming language.

## Tech stack
- automake
- cmake
- make
- ninja
- python

## Docker stack
- ubuntu:22.04

## Build note
This takes 1+ hrs to build.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
